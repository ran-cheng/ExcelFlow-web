API Key 申请指南

更新于 2025-04-02

本指南将帮助您申请 DeepSeek 和 SiliconFlow硅基流动 的 API Key，以便在 ExcelFlow 中使用高级 AI 功能（当然，您也可以使用其他的类OpenAI API）。请按照以下步骤操作。

DeepSeek API 申请指南

1

登录 DeepSeek 官网 API 开放平台

打开浏览器，访问 https://platform.deepseek.com/ 官方网站。进行注册/登录账号。

按照官网指引，使用手机验证码、或微信扫码可以进行注册和登录。
2

实名认证和充值

登录成功后，点击左侧的“充值”选项。

点击“去认证”按钮进行实名认证，而后选择金额并点击“去支付”按钮完成充值。
3

创建新的 API Key

点击左侧的“API keys”选项，创建新的密钥。

点击“创建API key”按钮，输入任意的 API key 名称，完成密钥的创建。
4

复制并保存 API Key

系统生成 API Key 后，立即复制并安全保存。注意：API Key 只会显示一次，请务必妥善保存！

立即复制并安全保存您的 API Key，它只会显示一次

SiliconFlow API 申请指南

1

登录 SiliconFlow 官网 API 开放平台

打开浏览器，访问 https://cloud.siliconflow.cn 官方网站。进行注册/登录账号。

按照官网指引，使用手机验证码、或微信扫码或其他方式进行注册和登录。
2

创建新的 API Key

点击左侧的“API 密钥”选项，创建新的密钥。

点击“新建 API 密钥”按钮，输入任意的描述信息，完成密钥的创建。
3

复制并保存 API Key

系统生成 API Key 后，立即复制并安全保存。

将鼠标移动至密钥上，点击鼠标进行复制。

重要提示

  • API Key 属于敏感信息，请勿分享给他人或发布到公开场合
  • 建议定期更换 API Key 以提高安全性
  • 如遇到 API 调用问题，请查看各平台的 API 文档或联系客服
